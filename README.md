Developed an interactive Power BI dashboard to analyze Zomato restaurant data across multiple countries and cities. The dashboard provides insights into restaurant distribution, average costs, ratings, cuisines, and customer votes to help understand dining trends and customer preferences.

📊 Dashboard Highlights
Total Restaurants: 9,551
Cities Covered: 141
Total Votes: 1 Million+
Average Aggregate Rating: 2.67

📈 Key Insights
Analyzed the average cost for two people across different cities.
Compared restaurant distribution by city using pie charts.
Evaluated average ratings and restaurant counts across multiple countries.
Identified the most popular cuisines based on customer votes.
Built an interactive matrix to compare cuisines with different currencies.
Added dynamic slicers for City and Country to enable user-driven analysis.
